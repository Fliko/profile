codeUp

codeUp is an application that uses your GitHub data to help create a profile for you. You can then search for users with similar interests, contact other users, create events that others can attend, and search for events in your area. This application was built using a React / Redux front end, NodeJS server, and MongoDB database. It was built in three weeks in an Agile team of five engineers utilizing a continuous integration environment where code was tested before merged on TravisCI, pulled from GitHub by Docker Cloud and served on a Digital Ocean Server. You can view it on GitHub at https://github.com/Henriettas-Army/CodeUp

Balance

Balance was the first application I built from scratch with a team, it is a fitness Journal that helps motivate you along your fitness journey. We used AngularJS, NodeJS server, and a MySQL database with knex. I had a hand in just about everything for this app and there is a lot to it. We made the mistake of wanting to add every feature on the internet to this app, fullcalendar.io, google graphs, owlCarousel and MaterializeCss, and we wanted more. All this while having a one-week deadline with all engineers working remotely! The most valuable skills I got from this app were time management and team communication, but it was also a great experience working with a full MVC framework like angularJS.

Aframe-Gallery

Aframe is an HTML framework for building VR experiences! I had been wanting to build an art gallery website for a while because of an awesome app a friend of mine built, an art gallery. I thought that I could have a lot of fun building it, but I wanted to use a new technology. Then I found Aframe which allows you to create 3d objects as HTML elements and has a great community as well with a lot of people sharing their creations. This gives you the benefit of being able to use javascript to manipulate the object, but they also have their ways of manipulation as well. This was not very difficult to learn and I encourage everyone to play around with it and maybe try to integrate it into one of your apps. This photo gallery was one of the most fun apps that I have built.
